# On-prem-to-Cloud-Data-Migration-using-Azure-Data-Factory
This project demonstrates an end-to-end data migration and transformation pipeline using Azure Data Factory, where data is migrated from:

🖥 On-Premises Files (CSV / Excel)

🗄 Azure SQL Database

☁ Loaded into Azure Data Lake Storage Gen2

🔄 Transformed into Silver Layer

📊 Aggregated into Gold Layer (Business Views)
